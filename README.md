# CalendarComponent
Данный компонент принимает на вход значение даты, например, '04/05/2021' или диапазона дат ['04/05/2021','15/05/2021'], а так же параметр формата даты, например 'DD/MM/YYYY'

Компонент форматирует даты в нужный формат, с которым работает v-date-picker, устанавливает текущую, если дата невалидна.
